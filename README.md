# 🚗 Vehicles Permits System

A system designed to manage vehicle permits efficiently, including request submission, approval workflows, and tracking.

---

## 📌 Description

The Vehicles Permits System helps organizations handle vehicle permit processes بسهولة and in an organized way.

It allows:
- Submitting permit requests
- Reviewing and approving permits
- Tracking permit status
- Managing vehicle records
